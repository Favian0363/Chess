# Chess Game

A basic chess game built with Pygame. (practicing OOP) 
Pieces can be clicked, dragged, and dropped onto squares, and the board renders using sprites. 
The project is a work-in-progress and currently includes early move-validation logic.
Current state:

Board rendering - Done
Piece placement - Done
Drag and drop - Done 
Coordinate conversions - Done
Move validation - In Progress
No turn logic, check or capture yet - To do
